Installation Guide

Up and running in 30 seconds.

Chrome extensions loaded from a zip require a quick manual install. Follow these four steps.

1
📦
Unzip the download
Find moment-extension.zip in your Downloads folder and extract it. You'll get a folder called moment-extension. Keep it somewhere permanent — Chrome needs it to stay there.
2
🌐
Open Chrome Extensions
In a new Chrome tab, navigate to chrome://extensions and press Enter. This is Chrome's built-in extension manager.
3
🔧
Enable Developer Mode
In the top-right corner of the Extensions page, toggle on Developer mode. This unlocks the ability to load local extensions.
4
✅
Load the folder
Click Load unpacked and select the moment-extension folder you extracted. Moment will appear in your extensions list — pin it to your toolbar and you're done.
