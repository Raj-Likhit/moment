Installation Guide

Up and running in 30 seconds.

Chrome extensions loaded from a zip require a quick manual install. Follow these four steps.

1
📦
Unzip the download
Find moment-extension.zip in your Downloads folder and extract it. Keep the folder somewhere permanent — Chrome needs it to stay there.
2
🌐
Open Chrome Extensions
In a new Chrome tab, navigate to chrome://extensions and press Enter.
3
🔧
Enable Developer Mode
Toggle on Developer mode in the top-right corner of the Extensions page.
4
✅
Load the folder
Click Load unpacked and select the moment-extension folder. Pin it to your toolbar and you're done.
